# 📊 دليل استخدام مجموعات البيانات

## المجموعات المتاحة

### 1️⃣ **BEIR/Fever** (5.4 مليون وثيقة)
- **الوصف:** مجموعة بحثية للتحقق من الحقائق
- **المحتوى:** ويكيبيديا ومصادر متنوعة
- **الاستخدام:** تحسين دقة البحث عن الحقائق والمعلومات المتحققة

### 2️⃣ **MSMARCO-Document** (3.2 مليون وثيقة)
- **الوصف:** وثائق حقيقية من الويب
- **المحتوى:** وثائق طبيعية بتنوع عالي
- **الاستخدام:** محرك بحث عام على الويب

---

## 🚀 طريقة الاستخدام

### تحميل مجموعة البيانات

**الأمر الأساسي:**
```bash
python main.py <dataset> [limit]
```

**الخيارات:**
- `fever` أو `1`: تحميل BEIR/Fever
- `msmarco` أو `2`: تحميل MSMARCO-Document
- `limit` (اختياري): عدد الوثائق المراد معالجتها (افتراضي: 200,000)

### أمثلة عملية

**📌 مثال 1: تحميل 200 ألف وثيقة من BEIR/Fever**
```bash
python main.py fever 200000
```

**📌 مثال 2: تحميل 300 ألف وثيقة من MSMARCO-Document**
```bash
python main.py msmarco 300000
```

**📌 مثال 3: تحميل الحد الأدنى (200 ألف) من BEIR/Fever**
```bash
python main.py fever
```

## 💾 ذاكرة التخزين المؤقت (Cache)

كل مجموعة بيانات لها:
- ملف بيانات JSON: `processed_data_<name>.json`
- ملف ذاكرة تخزين مؤقت: `cache/engine_cache_<name>.joblib`
- ملف بيانات وصفية: `cache/engine_cache_<name>_meta.json`

هذا يسمح بـ:
- ✅ تحميل سريع للمرة الثانية
- ✅ التحويل بين مجموعات مختلفة بسهولة
- ✅ مقارنة الأداء بين المجموعات
